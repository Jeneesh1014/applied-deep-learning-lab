# Applied Deep Learning Lab (SS26)

This repository contains my practical work and exercises for the **Applied Deep Learning** course (Summer Semester 2026) at Hof University of Applied Sciences.

## 📚 Course Overview

In this lab, I learn and implement the fundamentals of deep learning, including:

- Python programming basics
- Data handling using NumPy and Pandas
- Data preprocessing (splitting, normalization, standardization)
- Introduction to Neural Networks (classification & regression)
- Forward and backpropagation
- Hyperparameter tuning and optimization techniques
- Convolutional Neural Networks (CNNs)
- Basics of Deep Reinforcement Learning

## 🧪 Lab Work Structure

The repository is organized as follows:

- `notebooks/` → Jupyter notebooks for experiments and practice
- `scripts/` → Clean Python implementations
- `data/` → Datasets (not included in repo if large)
- `requirements.txt` → Python dependencies
